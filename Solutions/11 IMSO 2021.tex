\begin{proof}[\textbf{Solusi. }] Definisikan titik $E$ di dalam segiempat $ABCD$ sedemikian sehingga $AEBC$ membentuk jajargenjang.
\input{Asymptote/11 (2) IMSO 2021.tex}
Perhatikan bahwa $\angle DCE = \angle DCB - \angle ECB = \angle DCB - (180^\circ - \angle CBA) = 105^\circ - (180^\circ - 135^\circ) = 60^\circ$. Lalu, perhatikan bahwa $EC=AB=DC$. Ini berarti $\triangle EDC$ adalah segitiga sama sisi. Sekarang, $\angle DEA = 360^\circ - \angle AEC - \angle CED = 360^\circ - \angle CBA - \angle DCE = 360^\circ - 135^\circ - 60^\circ = 165^\circ$. Tetapi kita tahu bahwa $AE=BC=CD=DE$, maka $\angle ADE = \frac{1}{2}(180^\circ-165^\circ)=\frac{15^\circ}{2}$. Oleh karena itu didapat $\angle ADC = \angle ADE + \angle EDC = \frac{15^\circ}{2}+60^\circ = \frac{135^\circ}{2}$.
\end{proof}
